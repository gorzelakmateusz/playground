# test

TypeScript project created automatically.

## Available commands

```bash
# Run in development mode
pnpm run dev

# Build project
pnpm run build

# Run built project
pnpm run start

# Clean dist folder
pnpm run clean
```

## Project structure

```
test/
├── src/
│   └── index.ts
├── dist/           # Compiled files
├── package.json
├── tsconfig.json
└── README.md
```
